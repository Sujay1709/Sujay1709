### 📄 DocuMind — RAG Document Q&A Pipeline

[![Readme Card](https://github-readme-stats.vercel.app/api/pin/?username=Sujay1709&repo=documind-rag&theme=nord&border_color=0a2472&title_color=0a2472&icon_color=0a2472)](https://github.com/Sujay1709/documind-rag)

### 🧪 Model Regression Detection System

[![Readme Card](https://github-readme-stats.vercel.app/api/pin/?username=Sujay1709&repo=Model-Regression-Detection-System&theme=nord&border_color=0a2472&title_color=0a2472&icon_color=0a2472)](https://github.com/Sujay1709/Model-Regression-Detection-System)

Detects behavioral regressions in an LLM-based customer-support email classifier when prompts or models change — a 4-axis diff detector with CI-ready exit codes, versioned prompts, and a 75-case hand-curated golden dataset.

| Layer | Technology |
|---|---|
| LLM | Gemini 2.5 Flash (via `google-genai` SDK) |
| Embeddings | gemini-embedding-001 |
| Eval Harness | Python |
| Testing | Pytest (16 offline tests) |
| Config | YAML (externalized thresholds) |

💻 [**Code**](https://github.com/Sujay1709/Model-Regression-Detection-System) *(no live demo — CLI / CI tool)*

### 🚗 AutoHub — AI-Powered Automotive Platform

[![Readme Card](https://github-readme-stats.vercel.app/api/pin/?username=Sujay1709&repo=Auto_News_Site&theme=nord&border_color=0a2472&title_color=0a2472&icon_color=0a2472)](https://github.com/Sujay1709/Auto_News_Site)

A living automotive platform with a natural-language AI car assistant, side-by-side comparisons across 24 vehicles, interactive 3D models, and a live industry news feed — shipped as a pure front-end SPA.

| Layer | Technology |
|---|---|
| Frontend | React 18 + Vite + Tailwind CSS + Framer Motion |
| Routing | React Router (SPA) |
| 3D Viewer | `<model-viewer>` (GLB models) |
| News API | GNews.io via same-origin proxy |
| Hosting | Google Cloud Run (nginx + Docker) |
